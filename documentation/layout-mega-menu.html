Add sub navigation

You can add the Mega menu by default just adding the nav bt-sub-header between the bt-top-header and bt-content wrapper element, like this:

<html lang="en">
<body>
  <div class="bt-wrapper bt-mega-menu">
    <nav class="navbar navbar-expand fixed-top bt-top-header">
      <!-- Top bar -->
    </nav>

    <nav class="navbar navbar-expand-lg bt-sub-header">
      <!-- Mega menu -->
    </nav>

    <div class="bt-content">
      <!-- Main content -->
    </div>
    <div class="bt-right-sidebar">
      <!-- Right sidebar -->
    </div>
  </div> 
</body>
</html>

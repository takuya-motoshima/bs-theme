Remove sidebar left

You can remove the Left Sidebar by default just adding the class bt-nosidebar-left into the main wapper element, like this:

<html lang="en">
<body>
  <div class="bt-wrapper bt-nosidebar-left">
    <nav class="navbar navbar-expand fixed-top bt-top-header">
      <!-- Top bar -->
    </nav>
    <div class="bt-content">
      <!-- Main content -->
    </div>
    <div class="bt-right-sidebar">
      <!-- Right sidebar -->
    </div>
  </div> 
</body>
</html>

Remove both sidebars

Like the previous examples you can remove Both Sidebars elements directly from the html structure (bt-left-sidebar and bt-right-sidebar), additionaly you must to add the class bt-nosidebar-left into the main wapper element.

<html lang="en">
<body>
  <div class="bt-wrapper bt-nosidebar-left">
    <nav class="navbar navbar-expand fixed-top bt-top-header">
      <!-- Top bar without sidebar toggle buttons -->
    </nav>
    <div class="bt-content">
      <!-- Main content -->
    </div>
  </div> 
</body>
</html>
